Overview

This browser extension is designed to perform a single local function: visualize audio from the user's current browser tab. It does not collect, transmit, store, or share any personal information. All processing occurs entirely on the user's device.

Information Collection and Use

The extension does not collect or process any of the following:

  • Personally identifiable information
  • Authentication data or passwords
  • Location data or IP addresses
  • Financial or payment details
  • Browsing history or website content
  • User activity, input, or communications

The extension uses only standard browser APIs necessary to perform its function, including:

  • tabCapture - to obtain the current tab's audio stream
  • activeTab - to ensure the extension operates only on the user's selected tab

Captured audio remains in system memory and is never sent to external servers.

Data Sharing

No data is shared with third parties. No analytics, telemetry, advertising networks, or tracking scripts are included.

Open Source Transparency

The full source code is publicly available at:
https://github.com/Ronnie-Reagan/AudioVisualizerExtension/

Children's Privacy

This extension is suitable for general audiences and does not knowingly collect data from anyone, including children under 13.
